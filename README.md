# Language Translation Tool

This project is developed as part of CodeAlpha Artificial Intelligence Internship.

## Features
- Translate text between multiple languages
- Simple and clean UI
- Built using Python and Streamlit

## Technologies Used
- Python
- Streamlit
- Deep Translator

## How to Run
1. Install required libraries
2. Run `streamlit run translator.py`
